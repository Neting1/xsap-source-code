# Xeonsys Stock Analytics Platform (XSAP)

## Overview

XSAP is an AI-powered stock analytics and portfolio intelligence platform designed to help investors track, analyze, and monitor stocks across global markets.

## Technology Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python

### Database
- Firebase Firestore

### AI
- Google Gemini API

### Development Tools
- VS Code
- GitHub
- XAMPP
